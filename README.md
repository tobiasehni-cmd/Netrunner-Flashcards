# Netrunner Flashcards

Spaced repetition flashcard trainer for Android: Netrunner cards.

## Setup

1. Clone this repo
2. Enable GitHub Pages (Settings → Pages → Source: main branch)
3. Visit your site at `https://yourusername.github.io/netrunner-flashcards`
4. On iPhone: Safari → Share → Add to Home Screen

## Card Sets Included

- System Gateway (`sg`)
- Elevation (`elev`)
- Vantage Point (`vp`)

## How It Works

- Card data is fetched from the [NetrunnerDB public API](https://netrunnerdb.com/api/2.0/doc)
- Progress is stored locally in your browser via IndexedDB
- Works offline after initial sync
